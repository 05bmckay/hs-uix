# v0.1.0 — Initial Release

First published version of hubspot-datatable.

- Full-text search across configurable fields
- Select, multi-select, and date range filters with active filter chips
- Click-to-sort headers with three-state cycling
- Client-side and server-side pagination
- Collapsible row grouping with per-column aggregation
- Row selection with select-all
- Two inline edit modes (discrete click-to-edit, always-visible) with 10 input types and per-column validation
- Auto-width column sizing from data analysis
- Configurable row count display, bordered/flush appearance
- Server-side mode with loading/error/empty states, search debounce, controlled state, and onParamsChange callback
- Footer rows computed from filtered data
